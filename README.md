## What It Does

The BMIDE Analyzer lets Teamcenter engineers and PLM architects instantly analyze, document, and validate BMIDE configurations using Claude AI. Paste any BMIDE content and get expert-level insights in seconds.

### Analysis Modes

| Mode | Description |
|------|-------------|
| 🔍 **Explain Schema** | Plain-English explanation of any BMIDE artifact |
| 📄 **Generate Docs** | Auto-generate technical documentation |
| ⚡ **Impact Analysis** | Understand downstream effects of changes |
| ✅ **Validate Config** | Check against Teamcenter best practices |
| 💬 **Ask Anything** | Free-form Q&A about your BMIDE content |

### Supported Input Types

- Business Object (BO) definitions
- Property descriptors
- List of Values (LOV) configurations
- Workflow process templates
- Dataset type definitions
- Relation type definitions
- Custom attribute XML
- BMIDE export fragments

## How to Use

1. Open the [live demo](https://YOUR-USERNAME.github.io/bmide-analyzer/)
2. Enter your **Anthropic API key** (get one at [console.anthropic.com](https://console.anthropic.com))
3. Paste your **BMIDE content** (or use a sample)
4. Select an **analysis mode**
5. Click **Analyze Now**

> 🔒 Your API key is stored in memory only and never sent to any server other than Anthropic's API.
